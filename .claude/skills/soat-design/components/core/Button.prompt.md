Brand-gradient call-to-action button; use for primary actions, secondary for bordered alternatives, ghost for low-emphasis inline actions.

```jsx
<Button variant="primary" size="lg">Get Started — 5 min</Button>
<Button variant="secondary">Read the docs</Button>
<Button variant="ghost" size="sm">Cancel</Button>
```

Variants: `primary` (gradient + hover lift/glow), `secondary` (surface + border, border turns cyan/blue on hover), `ghost` (text only). Sizes: `sm` / `md` / `lg`. Accepts `iconLeft` / `iconRight` nodes and standard button props.
