Compact pill for statuses and labels; tonal background tinted from the semantic palette, with an optional status dot and a "glow" tone for dark-mode emphasis.

```jsx
<Badge tone="success" dot>healthy</Badge>
<Badge tone="glow">MCP native</Badge>
<Badge tone="neutral">v1.1</Badge>
```

Tones: `neutral`, `primary`, `success`, `danger`, `warning`, `glow`. Set `dot` to prepend a status indicator.
