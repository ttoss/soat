Raised surface panel. `solid` for standard content, `glass` for blurred HUD panels over imagery, `interactive` to lift + glow on hover (feature/surface grids).

```jsx
<Card>
  <h4>REST API</h4>
  <p>OpenAPI-described HTTP endpoints for any backend.</p>
</Card>

<Card variant="glass" interactive>…</Card>
```

Props: `variant` ("solid" | "glass"), `interactive`, `padding`.
